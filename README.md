# AppMachine Training: CodeLab - Caching with progressive libraries Design

This is a project that is part of the training for the selection process of the enterprise App Machine. This project was implemented the use of sw libraries-precache and sw-toolbox in a web app.

## Requirements:

Have installed on the machine the Chrome browser in version 49 or higher, a web service and the node.js with npm and gulp (also installed).

## Contributions:

Only a developer participated in this project. The programmer Felipe Mitterofhe.

## Version control:

The system is currently in version 1.0. This version includes the implementation of the code that adds the libraries sw-precache and sw-toolbox in web app.
